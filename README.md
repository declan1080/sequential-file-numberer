# Sequential File Numberer

This .BAT script renames files in sequential order.

## Features

- Rename in current directory.
- Rename in custom directory.
- Optionial prefix.
- File type filtering.
- Custom starting number.
- Changes preview.
- Checks for write access and path existence.
